# A121 + nRF52840 Radar Module -- Session Report

## 1. System Overview
```text
-   MCU: nRF52840-QIAA-R7 (BLE / Thread / Zigbee)
-   Radar: Acconeer A121 60GHz
-   Main Power Rail: 1.8V generated by TPS62840 buck
-   VIN monitoring: SIP32431 + resistor divider
-   Application: Near presence detection (30--50 cm), 10 Hz radar raw
    frames
```
------------------------------------------------------------------------

## 2. Power Consumption Summary

```text
  Block                Typical Average
  -------------------- -----------------------------
  A121 Radar           3--6 mA @1.8V
  nRF52840 + BLE       2--4 mA @1.8V
  VIN_ADC Divider      \~0.02 mA
  TPS62840 Quiescent   \~0.00006 mA
  Margin               1--2 mA
  **Total Avg**        **\~10 mA @1.8V (\~0.02W)**
```

-   Peak transient current during radar burst: **150--200 mA**
-   Average power: **\~20 mW**

## 6. RF Artwork Key Rules
```text
-   4-layer PCB minimum (L1 RF / L2 solid GND / L3 power / L4 signals)
-   2.4GHz trace 50Ω microstrip, no vias, \<15mm
-   A121 requires solid uninterrupted GND plane under package
-   Via fence 0.5--1mm around A121 and RF paths
-   Antenna keep-out ≥10mm radius, no copper
```
------------------------------------------------------------------------

## 7. Final Notes
```text
-   Design rail for **10mA average / 200mA transient**
-   A121 layout and GND padding are mandatory for mmWave stability
-   Battery life mainly driven by duty cycle of radar bursts
```
